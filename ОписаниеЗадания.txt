:: ТЕСТОВОЕ ЗАДАНИЕ ::

Создать консольную программу, которая работает с базой данных.

Условия и описание задания:
- Программа должна быть написана на С++
- Программа должна получать в параметрах командной строки расположение базы данных (путь до файла, название файла)
- Файл базы данных всегда имеет расширение sq3 (SQLite)
- В базе содержится таблица passenger. Из нее следует извлечь все данные из столбцов DateTime, OriginZone, DestZone. 
- Данные следует отсортировать по столбцу DateTime
- Вывести данные в консоль в виде таблицы
- Записать данные в текстовый файл, который располагается в том же каталоге, что и база данных, имеет то же имя, что и база данных, но расширение у файла txt.
- Всю работу с файлом базы данных следует делать через официальную библиотеку sqlite3 (https://www.sqlite.org/download.html)
- Тестовый файл базы данных приложен

Пришлите, пожалуйста, в ответ:
- скомпилированную программу
- проект с исходным кодом
- результат выполнения программы с тестовой базой данных (файл txt)

Спасибо.
